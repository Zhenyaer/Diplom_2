# Что было сделано в рамках задания:
# 1. Написаны тесты на проверку создания пользователя test_user_create
# 1.1. Проверка создания уникального пользователя
# 1.2. Проверка невозможности создания уже зарегистрированного пользователя
# 1.3. Проверка невозможности создания пользователя при отсутствии одного из обязательных полей
# 2. Написаны тесты на проверку логина пользователя test_user_login
# 2.1. Проверка авторизации существующего пользователя
# 2.2. Проверка ошибки авторизации при неверном логине/пароле
# 3. Написаны тесты на проверку изменения данных пользователя test_user_change_data
# 3.1. Проверка изменения данных авторизованного пользователя
# 3.2. Проверка возникновении ошибки при попытке изменения данных неавторизованного пользователя
# 4. Проверка создания заказа test_order_create
# 4.1. Проверка создания заказа с ингридиентами авторизованным пользователем
# 4.2. Проверка создания заказа с ингридиентами неавторизованным пользователем
# 4.3. Проверка ошибки при создании заказа без ингридиентов
# 4.4. Проверка ошибки при создании заказа c неверным хэшем ингридиентов
# 5. Проверка получения списка заказов пользователя test_order_receive
# 5.1. Проверка получения списка заказов авторизованного пользователя
# 5.2. Проверка ошибки получения списка заказов неавторизованного пользователя
#
# Используемые библиотеки: pytest, requests, allure-pytest
# Запуск тестов: pytest -v
# Запуск визуализации отчета выполнения тестов: allure serve allure_results
